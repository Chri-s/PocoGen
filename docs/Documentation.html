PocoGen consists of two applications, PocoGen.GUI and PocoGen.Runner. PocoGen.GUI is the primary application to configure the database source, naming rules and output formats. All these information make a Poco definition which can be executed to generate the output files and also be saved.

PocoGen.Runner is a command line application which just executes the Poco definitions. This is handy to recreate the output during a build in a Pre-Build command in Visual Studio, just like T4 templates.
