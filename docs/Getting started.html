The easiest way to start with PocoGen is to download and execute the setup. Additional to copying the application files to your computer, it also associates *.poco files with PocoGen.GUI and adds the path of PocoGen.Runner to the App Paths.
That allows us to use the PocoGen.Runner in a Visual Studio solution as Pre-Build command without specifying the complete path in it. This way it can be executed on different computers having it installed in different directories.
