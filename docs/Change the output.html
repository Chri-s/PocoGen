After creating the first POCO you should have one output writer, “POCOs” with an empty filename. Enter a filename relative to the poco definition you saved at the end of creating the first POCO.

That means, if the output file should be at the same folder as the poco definition, just enter the filename (1). If the output file should be in a subfolder, for example “Model” in the picture, enter the subfolder and filename “Model\POCOs.cs” (2). And if the output should be one directory above the poco definition, enter “..\POCOs.cs” (3).

Now you can generate the output just by clicking on “Generate Output” in the toolbar.
