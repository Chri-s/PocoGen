Project Description
Extensible tool to automatically create POCOs and mappings for different databases and ORMs. Apply rules to create the class and property names.

 

PocoGen enables you to easily create your POCOs and mappings in a graphical interface, without the need to edit T4 templates. But just like T4 templates in Visual Studio, the code can be automatically created each time you build your project.

Currently these databases are supported:

  • Microsoft SQL Server
  • MySQL
  • PostgreSQL
  • SQLite

PocoGen is completly modular. You can add other databases by implementing a simple interface and adding the assembly to the installation folder.

Unlike T4 templates, you can easily apply naming rules to your table and column names like creating the class UserRole for the table user_roles. You can also manually rename your classes and properties or exclude them from the generated output.

The included code generators support C# and VIsual Basic.

 

PocoGen requires the .NET-Framework 4.5 installed. Connection-Libraries for the databases are included.
